###과끼리### <br/>
전공에 대한 정보 게시판 <br/>
일반적인 전공 질문 게시판같은 경우에는 상대방이 전공자인지 확인이 어렵지만 <br/>
‘과끼리’같은 경우에는학교 인증을 통하여 전공자인지 확인할 수 있다.

### 참여한 부분
+ 게시판 API 제작
+ 로그인 API 관련 서비스
+ Redis를 활용한 도배방지 시스템
+ WAS server 구축
