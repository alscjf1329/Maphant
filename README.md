**과끼리**

> ‘과끼리’는 전공에 대한 정보 게시판
> 

일반적인 전공 질문 게시판같은 경우에는 상대방이 전공자인지 확인이 어렵지만 ‘과끼리’같은 경우에는학교 인증을 통하여 전공자인지 확인할 수 있다.
